# Furina Language (v0.1)

Furina is a custom programming language implemented from scratch in Python.

## Features
- Custom grammar and syntax
- Lexer (tokenization)
- Recursive descent parser
- Abstract Syntax Tree (AST)

## Project Structure
lexer.py       # lexical analysis
parser.py      # syntax analysis (AST construction)
ast_nodes.py   # AST node definitions

docs/          # language design documents
examples/      # sample Furina programs
